
# Login Test Plan

**Goal:** Make sure users can log in correctly.

**What we will test:**
- Login with correct email/password
- Login with wrong email/password
- Login with empty fields

**How to test:** Manual testing

**Who will test:** 1-2 testers

**Where to test:** Chrome and mobile

**Steps to follow:**
1. Open login page
2. Enter email and password
3. Click Login
4. Check if it works or shows error message

**Done when:**  
All login tests work and no major bugs remain.
