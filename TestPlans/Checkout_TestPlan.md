
# Checkout Test Plan

**Goal:** Make sure users can buy products successfully.

**What we will test:**
- Fill in address
- Choose payment method
- Place order
- See order confirmation
- Handle errors if payment fails

**How to test:** Manual testing

**Who will test:** 1-2 testers

**Where to test:** Chrome and mobile

**Steps to follow:**
1. Open cart and click checkout
2. Enter address and payment details
3. Click Place Order
4. Check order confirmation page
5. If payment fails, check error message

**Done when:**  
Checkout works and order confirmation is shown correctly.
